# FnKismetDecompiler
Kismet decompiler for fortnite

# Unreal Version Support
Versions below the unreal versions are fortnite builds i've tested it on
 - UE4.20 - Supported
   - 4.1
 - UE4.22 - Supported
   - 7.30
 - UE4.23 - Supported
   - 8.51
   - 10.40
 - UE4.25 - Supported
   - 12.41
 - UE4.26 - Supported
   - 14.60
   - 18.40
 - UE5.0-EA - Supported
   - 19.40
 - Everything else = Not tested

# Goal
The final goal is to decompile kismet bytecode to psuedo c++ on unreal versions 4.20 to 5.4 (maybe more in the future but rn it's the only versions i care about)
